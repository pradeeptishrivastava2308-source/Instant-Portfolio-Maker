# Instant-Portfolio-Maker
Instant Portfolio maker is a website, that allow students to generate their portfolio instantly.
Many students maintain their resume, but they don't have their personal portfolio. Since portfolio requires technical knowledge and deployment knowledge, students either have incomplete portfolio or they never make it.
Instant Portfolio maker allows users to edit their details, and generate their portfolio instantly by providing ready made templates. Students can select the template of their choice and enter their details and save it using save button.
As of now, I have made frontend part, working on the backend to make it more user-friendly.
